# Business Analytics Data Analysis
This project analyzes a dataset of 10,000 business transactions using Python.

## Features
- **Data Cleaning:** Handled dates and verified 0 missing values.
- **EDA:** Analyzed Revenue and Profit across Regions and Product Categories.
- **Visualizations:** Created Bar charts for Payment Methods and Line charts for Monthly Trends.

## Libraries Used
- Pandas
- Matplotlib
- Seaborn
